## Project Description
ScamSniper AI is an intelligent cybersecurity tool that helps users detect phishing and scam messages in real time. Users can paste any suspicious message, email, or text into the system, and the AI analyzes it to determine whether it is safe or potentially harmful.

The system uses AI models to identify patterns commonly found in phishing attempts, such as urgency, suspicious links, and deceptive language. It then provides a risk level (Low, Medium, High) along with an explanation to help users understand why the message may be dangerous.

This tool is designed to improve digital safety by making threat detection simple, fast, and accessible to everyone.
